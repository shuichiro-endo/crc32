# crc32
get crc32 hash

## Usage
1. download the latest [crc32](https://github.com/shuichiro-endo/crc32)
```
git clone https://github.com/shuichiro-endo/crc32.git
cd crc32
```
2. build
```
gcc crc32.c -o crc32
```
3. run
```
usage   : ./crc32 function_name
example : ./crc32 LoadLibraryA
```
